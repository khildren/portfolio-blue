Blue Solutions — Portfolio Admin Guide

Everything you need to manage the Living Portfolio. Add a project in Google Drive and it shows up here automatically. Plain English, step by step.

📁 Living Portfolio — Google Drive Workflow
How it works: You manage projects entirely from Google Drive. Create a folder, name it with the right format, drop in images and text — the site updates automatically on the next sync. No coding, no uploading to the site directly.
➕

Add a New Project

  1. Open the Portfolio Google Drive folder
  2. Create a new folder — name it using the format below
  3. Drop in your images (see naming tips)
  4. Optionally add a description.txt file
  5. The site will pick it up on the next sync
  6. To sync right now: use the admin action below
View Projects →
🏷️

Folder Naming Format

  1. Use pipes ( | ) to separate the fields
  2. Only the Project Name is required
  3. Fields in order: Name | Category | Location | Year
📂 Hillside Kitchen | Residential | Boulder CO | 2024 📂 Downtown Office | Commercial | Denver | 2023 📂 Mountain Retreat
🖼️

Image Naming — Hero First

  1. Name your hero / flyer image: cover.jpg, hero.jpg, or flyer.jpg — it becomes the project thumbnail
  2. Number the rest to set gallery order: 01_living.jpg, 02_kitchen.jpg
  3. Subfolders are fine — subfolder name becomes the image caption
  4. Supported: JPG, PNG, WebP
📂 Hillside Kitchen | Residential | Boulder | 2024 cover.jpg  ← hero image 01_living-room.jpg 02_kitchen.jpg description.md
📝

Add a Description

  1. Create a file called description.txt or description.md in the project folder
  2. Write the project description in plain text
  3. Use .md if you want bold, headings, or bullet points
  4. The sync will pick it up automatically
description.md
View Projects →
⚙️

Override Metadata with info.txt

  1. Drop an info.txt file in any project folder
  2. Use plain key: value lines to override fields
  3. Useful when you can't rename the folder (client-supplied names)
info.txt contents: featured: yes category: Residential location: Aspen, CO year: 2024
🔄

Trigger a Sync Now

  1. Go to Portfolio → Projects
  2. To sync one project: check its box → Actions → ↻ Sync selected
  3. To sync everything: check any box → Actions → ↻ Full sync ALL
  4. The sync reads only what changed since last time — it's fast
  5. Deleted Drive folders → project marked inactive (not deleted)
Go to Projects →
🗂️ Managing Projects in Admin
Drive is the source of truth for images and content. Use the admin to set display order, mark projects as featured, and tweak details the folder name can't express. Changes made here won't be overwritten by sync unless the Drive folder is also updated.
⭐

Feature a Project on the Homepage

  1. Go to Portfolio → Projects
  2. Find the project in the list
  3. Check the Featured checkbox in the list view
  4. Click Save — it appears in the homepage slider instantly
  5. Or drop featured: yes in an info.txt in Drive
Go to Projects →
↕️

Set Display Order

  1. Go to Portfolio → Projects
  2. The Order column is editable inline
  3. Lower number = shows first (0 is first)
  4. Change the numbers and click Save at the bottom
Go to Projects →
🖼️

Change the Cover / Hero Image

  1. Open the project in admin
  2. Scroll to the Cover Image field
  3. Select a different image from the dropdown
  4. Or — rename the image to cover.jpg in Drive and re-sync
🚫

Hide a Project

  1. Go to Portfolio → Projects
  2. Uncheck the Active checkbox for that project
  3. Click Save
  4. The project disappears from the site but stays in the database
  5. To permanently remove: delete the folder from Drive, then sync
Go to Projects →
